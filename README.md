# JS-and-React
## Типы данных
### Null и undefined
null - переменная отсутствует;

undefined - переменная есть, но она не содержит никакого значения.
### Массивы
Массив - частный случай объекта (ключ - индекс), не отдельный тип данных. Могут содержать любые типы данных.
